## Development Setup

Clone the repository:

```bash
git clone https://github.com/ako-mawlood/portfolio
```

Navigate to the folder:

```bash
cd portfolio
```

Install dependencies:

```bash
bun install
```

then

```bash
bun run dev
```

and
open the brower
http://localhost:3000
